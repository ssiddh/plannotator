      const { token } = useGitHub();
      const [isExporting, setIsExporting] = useState(false);

      const [lastResult, setLastResult] = useState<PRMetadataWithSync | null>(null);

    Main export function (stored via useCallback):
    `exportToPR(pasteId: string, planMarkdown: string, annotations: ExportAnnotation[], blocks: ExportBlock[], retryCount = 0): Promise<PRMetadataWithSync | null>`

    Implementation:
    - Set `isExporting = true`, `error = null`
    - Per D-19: if `!token`, set error to "Authentication required" and return null
    - `fetch("/api/pr/create", { method: "POST", headers: { "Content-Type": "application/json", Authorization: "Bearer " + token }, body: JSON.stringify({ pasteId, planMarkdown, annotations: annotations.map(a => ({ id: a.id, blockId: a.blockId, type: a.type, text: a.text, originalText: a.originalText, images: a.images })), blocks: blocks.map(b => ({ id: b.id, startLine: b.startLine })) }) })`
    - On response:
      - Per D-18: if `res.status === 429 && retryCount < 3`: extract `Retry-After` header or use `Math.pow(2, retryCount) * 1000` ms delay, `await new Promise(r => setTimeout(r, delay))`, return `exportToPR(pasteId, planMarkdown, annotations, blocks, retryCount + 1)`
      - Per D-19: if `res.status === 401`: clear token from localStorage (`localStorage.removeItem("plannotator_github_token")`), set error to "Authentication expired. Please sign in again.", return null
      - Per D-20: network errors (catch block): if `retryCount < 3`, delay with exponential backoff and retry. After 3 retries, set error.
      - On success (2xx): parse JSON as PRMetadataWithSync, set `lastResult`, return it
      - On other errors: parse error text, set error state, return null
    - Always: set `isExporting = false` in finally block

    Return: `{ exportToPR, isExporting, error, lastResult, retryCount }` where retryCount is tracked via a ref for UI display.

    Actually, for retry tracking in UI, add a `retryAttempt` state (number, starts at 0):
    - Set to `retryCount + 1` before each retry delay
    - Reset to 0 on success or final failure

    Final return type:
    ```typescript
    return { exportToPR, isExporting, error, lastResult, retryAttempt };
    ```
