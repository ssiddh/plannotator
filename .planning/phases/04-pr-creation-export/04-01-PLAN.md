       - If line comments exist OR global comments exist: call `submitBatchReview(owner, repo, prMetadata.pr_number, token, lineComments, globalBody)`
       - Per D-10: generate plan hash via `generatePlanHash(planMarkdown)`

       - Store to KV: `kv.put("sync:" + pasteId + ":pr", JSON.stringify(metadataWithSync), { expirationTtl: ttlSeconds || 30 * 24 * 60 * 60 })` (per D-09, D-11)

       - On failure after branch creation: iterate `createdResources.reverse()`, delete each branch via `githubRequest("DELETE /repos/${owner}/${repo}/git/refs/heads/${branchName}", token).catch(() => {})` (per D-17, pitfall 4: use plural `refs`)
       - Return `PRMetadataWithSync`
       - Per D-18/D-20: retry logic is NOT in this function (it's in the client hook, per research)
